# Slides da apresentação

A apresentação final foi construída em **Reveal.js** (HTML), seguindo a identidade
visual oficial do CIn-UFPE.

[:material-presentation-play: Abrir os slides em tela cheia](slides.html){ .md-button .md-button--primary target=_blank }
[:material-script-text-play: Guia de falas (ensaio)](estudo-falas.html){ .md-button target=_blank }

!!! tip "Como apresentar"
    - Abra `slides.html` no navegador e pressione **F** para tela cheia.
    - Use as **setas** (← →) para navegar; **Esc** mostra a visão geral.
    - Pressione **S** para abrir a *speaker view* com as **notas de apresentador**
      e um cronômetro (útil para controlar os 20 minutos).

## Estrutura (20 min · 4-5 apresentadores)

Cada slide tem um **dono sugerido** (baseado em quem liderou a área no projeto) e
o **item do roteiro do professor** que ele cobre.

ver, já terá acontecido.
